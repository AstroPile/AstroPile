# Spectral information schema

| Field 	| Unit 	| Description 	|
|-------	|------	|-------------	|
| flux      | erg/cm$^2$/$\AA$/s   	| Flux            	|
| lambda    | $\AA$ | Wavelengths   |
| ivar    	|      	| Inverse variance of the flux |
| lsf_sigma |       | Line spread function $\sigma$ / resolution |
